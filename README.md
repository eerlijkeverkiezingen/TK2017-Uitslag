Hier vind u de verzameling (gedigitaliseerde) **proces-verbalen** omtrent de Verkiezingen voor de [Tweede Kamer der Staten-Generaal](http://www.tweedekamer.nl/) van *15 maart 2017*. Deze verzameling is ontstaan door het onderzoek van [Eerlijke Verkiezingen](http://eerlijkeverkiezingen.nl/onderzoek-2017) in samenwerking met haar partners *(zie [persbericht](2017-03-30-persbericht.md))*.
